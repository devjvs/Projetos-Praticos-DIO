História dos computadores

Voltar

Bem, o primeiro computador a ser criado se chamava Eletronic Numerical Integrator and Computer — ENIAC, entre 1943 e 1946. Para desenvolvê-lo, foi investido cerca de 6 milhões de dólares, onde seu funcionamento era dependente de uma grande quantidade de fatores, como 200 mil watts de energia.

Ainda em 1993, surgiu a World Wide Web, mais conhecida como WWW. Tempos depois, começaram a surgir os smartphones. Do ENIAC para o primeiro smartphone, passaram-se cerca de 70 anos.

O que são Clients?

Voltar

Na arquitetura cliente-servidor, o dito client é aquele que sempre inicia a comunicação com o servidor, onde o mais conhecido é o navegador. No caso, quando acessamos um site pelo navegador, é feita uma requisição, que é disparada pela internet para o computador no qual esse site está hospedado.

O que são Servers?

Voltar

Diferente do Client, o papel do servidor é receber uma requisição e devolver uma resposta para o cliente, tendo como principal característica essa, onde o servidor não envia dados se o cliente não disparar primeiro uma requisição.

Mais especificamente, o servidor é um ou mais computadores com um único endereço, sendo possível hospedar documentos, imagens, áudio, vídeo, etc.

Como é uma estrutura básica de um HTML?

Voltar

A estrutura básica de um arquivo HTML se inicia por: 

        
            !DOCTYPE html
            html lang="pt-br"
            head
                title Título da página /title
                meta charset="utf-8"
            /head
            body
                Aqui vai o código HTML que fará seu site aparecer.
            /body
            /html

Entendendo a diferença entre listas ordenadas e não ordenadas

Voltar

Bem, a principal diferença entre a lista ordenada e a lista não-ordenada está na maneira com que os itens de cada lista são apresentados - a lista ordenada enumera os elementos; a lista não-ordenada coloca marcadores antes de cada elemento.
